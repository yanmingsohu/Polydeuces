# Polydeuces 

* Javascript 引擎
* 研究用项目, 不可用于生产

## 目标 

* 尽可能减少内存占用
* 支持系统原生线程
* 测试如果不执行任何优化其性能与专业引擎的差距

## 需要

* Windows 7+
* Java 1.7+


## install

1. 安装依赖, 参看: deps/Readme.md
2. 运行 generate-src.cmd 生成/更新 js 语法框架代码(谨慎).
3. 编译
